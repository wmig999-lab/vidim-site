
## Продукты

| URL | Описание | Технологии |
|---|---|---|
| [vidim.site](https://vidim.site) | Лендинг — 6 секций, матричная анимация | Vanilla HTML + CSS + JS |
| [brief.vidim.site](https://brief.vidim.site) | Бриф-опросник — 22 вопроса + ИИ-отчёт | React 18, localStorage |

## Бриф — режимы работы

**Демо-режим (по умолчанию):** отчёт собирается из шаблонов на основе ответов. Не требует API-ключа.

**Режим ИИ (для тестирования):** нажмите ⚙ в правом нижнем углу страницы брифа, вставьте Anthropic API-ключ. Отчёт генерирует Claude.

> ⚠ В production-версии API-ключ должен быть на сервере, не в браузере. Текущий режим — только для тестирования.

## Деплой

Репозиторий подключён к [Vercel](https://vercel.com). Каждый коммит в `main` автоматически публикует обновления.

**Домены в Vercel:**
- `vidim.site` → корень репозитория (`index.html`)
- `brief.vidim.site` → папка `brief/` (`brief/index.html`)
